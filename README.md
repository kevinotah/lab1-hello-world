# Hello World (Python)

A minimal Python starter project for AI4SE Lab 1.

## Overview

This project demonstrates a basic Python program structure using a `main()` function and an entry-point guard.

## Requirements

- Python 3.x

## How to Run

```bash
python main.py
```

## Expected Output

```text
Hello, World!
```

## Project Structure

- `main.py` - Entry point that prints `Hello, World!`
